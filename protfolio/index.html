About


Welcome to my website!
I'm a 22-year-old certified Front-End Developer, proudly trained by Meta Company (formerly known as Facebook). Through rigorous and comprehensive courses, I have honed my skills in various categories, including Cascading Style Sheets (CSS), HTML, UI/UX design, React, and JavaScript.

In the realm of web development, I possess the expertise to craft visually stunning and responsive websites. Using HTML to structure content, CSS to handle visual styles, and JavaScript to create engaging and interactive experiences, I am committed to delivering top-notch results that captivate users and meet modern web standards.
With a focus on mastering the React library in tandem with JavaScript frameworks, I am adept at building dynamic and efficient web applications. Additionally, I leverage the power of Bootstrap CSS Framework to create elegant and functional web pages, while also utilizing GitHub repositories and version control for seamless collaboration with teams.
In my pursuit of excellence, I have also delved into the principles of UX/UI design, understanding the importance of creating user-centric interfaces that enhance user experiences. This skill set allows me to create intuitive and delightful interactions that leave a lasting impression on users.
Throughout my learning journey, I have prepared extensively for coding interviews and cultivated a problem-solving mindset that enables me to tackle challenges with confidence. My dedication to building portfolio-ready projects ensures that I am well-prepared to showcase my abilities during job interviews.

On this website, you will find a professional certificate showcasing my achievements in a comprehensive 9-course series. Starting with an introduction to Front-End Development, I have delved deep into JavaScript programming, version control, and the intricacies of HTML and CSS. My journey culminates in advanced React concepts, where I have refined my skills to bring innovative ideas to life.
